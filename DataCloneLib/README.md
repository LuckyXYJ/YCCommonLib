### 01.基础概念介绍
#### 1.1 什么是数据拷贝
- 对象拷贝(Object Copy)就是将一个对象的属性拷贝到另一个有着相同类类型的对象中去。主要是为了在新的上下文环境中复用对象的部分或全部数据。


#### 1.2 三种类型拷贝
- Java中有三种类型的对象拷贝：浅拷贝(Shallow Copy)、深拷贝(Deep Copy)、延迟拷贝(Lazy Copy)。
